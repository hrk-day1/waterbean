## 표준 용어 (권장)

| 개념 | 권장 표기 | 비고 |
|------|-----------|------|

| 방문자용 크리에이터 페이지 | **공개 홈** / **크리에이터 홈** | `prof-PR2`, `home-*` |
| 로그인 없이 열람 | **전체 공개** | 홈 행·포스트 권한 공통. |
| 유료 티어(상품 연동) | **OO 구독** (베이직·프리미엄·VIP) | 상품명은 `mem-M2` 매핑표 따름. **O-11** 홈·포스트 모두 **구독** 표기. |
| 홈 큐레이션 행 가시성 라벨 | 예: **VIP 구독** | 포스트 열람 권한·상품명과 동일 **구독** 용어(O-11). |
| 큐레이션 묶음 | **큐레이션 그룹** | `home-H4`~`H6` |

| 저장 버튼(상단) | **변경사항 저장** | `shell-G3` |

## 일관성 점검 결과

- [x] **공개**: 전체 공개·그룹/행 공개·포스트 공개 설정이 혼동되지 않도록 각 조각에서 한글 **공개** 유지.
- [x] **구독 vs 멤버십**: 유료 티어 UI 문구는 **구독**(베이직·프리미엄·VIP 구독). **O-11**로 홈 가시성까지 동일. (내비·메뉴명 **멤버십 상품** 등 조직/기능명은 별개.)
- [x] **VIP**: 홈·포스트·상품 **VIP 구독**으로 통일 — **O-11** 확정, [`cross-open-issues.md`](cross-open-issues.md) 잔여 아님.
- [x] **미리보기 / 변경사항 저장**: `shell-G3` 판정표를 단일 기준으로 함.
