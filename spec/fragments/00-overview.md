# 00 — 개요 · 범위 · 용어 (최소본)

## 목적

크리에이터가 **공개 홈**에 보이는 커버·고정 포스트·큐레이션 구역을 설정하고, 포스트·멤버십·프로필을 관리하는 **Creator Center** 관리 UI에 대한 상세기능 명세의 기준점.

## 범위

- 포함: 와이어에 나타난 관리자 앱 영역(좌측 내비 + 본문).
- 제외(기본 가정): Figma 호스트 상단의 파일명·「이메일로 가입하기」「Google로 계속」·쿠키 배너는 **제품 기능이 아님**. 제품 내 계정/로그인은 별도 IA라면 `TBD`.

## 용어

| 용어 | 의미 |
|------|------|
| 홈(내비) | 관리 UI의 「홈」 메뉴 → 본 명세에선 **홈 큐레이션** 화면으로 매핑. |
| 큐레이션 그룹 | 예: 「인기 콘텐츠」처럼 제목이 있는 묶음. 하위에 포스트(또는 콘텐츠) 행이 속함. |
| 가시성 | 누가 해당 콘텐츠를 볼 수 있는지(예: 전체 공개, VIP 구독). 유료 티어는 **구독** 표기로 통일(O-11). |
